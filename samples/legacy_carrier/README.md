# Consolidated Mutual of Ohio — Legacy AS/400 Export

- **Format:** Pipe-delimited CSV, latin-1
- **Dates:** YYYYMMDD integers
- **Amounts:** Mostly cents; some `$` formatted strings
- **Quirks:** Trailing pipes, whitespace, `NULL` literals, YYYYNNNNN policy numbers

## Files

- `claims.csv` — 20 claims
- `exposures.csv` — 35 rows
- `contacts.csv` — 40 rows
- `transactions.csv` — 50 rows
- `data_dictionary.md` — Partial BA documentation
